La solution iFrame

Les médias mettent une dataviz sur leur site sans se poser de question. Elle est intégrée pour utiliser toute la place qui lui est permis.

Il est possble d'éviter la double barre de defilement (pas fait), mais si le site est responsive on risque de ne pas afficher tout son contenu...

Il y a quelques solutions pour régler le problème, mais c'est tout de même pas mal de soucis et de difficultés pour pas grand chose.

Sinon, un moyen simple d'éviter les problèmes de barres de défilement est de n'afficher que l'iframe dans la page (pas d'autre texte, donc pas comme dans cette page d'exemple) => cf. test3.html
